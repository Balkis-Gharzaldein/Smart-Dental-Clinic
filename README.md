Features

+ Smart appointment scheduling and clinic calendar management

+ Patient data management and medical history tracking

+ Staff and dentist management

+ Reports and analytics on patients and visits

+Panoramic X-ray image analysis for diagnosis assistance, including:

  + Bone level assessment
  + Detection of wisdom teeth and tooth buds
  + Teeth segmentation using U-Net model



Technologies Used

+ Programming Language: Python

+ Frameworks: Django

+ Database: MySQL

+ Libraries: OpenCV, Pandas, NumPy

+ Deep Learning Model: U-Net for teeth segmentation
